About Kenya JUG

Kenya Java User Group (JUG) is a vibrant community of Java developers, architects, and enthusiasts dedicated to advancing Java technology in Kenya. We provide a platform for knowledge sharing, professional networking, and skill development.

Our mission is to foster innovation, collaboration, and continuous learning within the Java ecosystem while supporting the growth of the local tech community.

Our Programs

Wed Weekly

LeetCode Wednesdays

Led by Cosmas Gikunju

Sharpen your problem-solving skills with our weekly LeetCode sessions. We tackle algorithmic challenges together, discuss optimal solutions, and prepare for technical interviews.

Thu Weekly

Java Enhancement Proposals (JEP) Thursdays

Led by Samuel Owino

Dive deep into the latest Java Enhancement Proposals and upcoming Java features. We discuss new language features, JVM improvements, and how they impact our development practices.

Sat 8:00 PM - 10:00 PM

Weekly Virtual Sessions

Virtual Event

Join our regular Saturday evening sessions covering various Java topics, frameworks, and best practices. We also have a weekly call for speakers - reach out if you'd like to share your knowledge!
